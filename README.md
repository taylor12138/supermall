# supermall
该项目是基于Vue.js开发的仿蘑菇街电商项目
实现了作为电商类app的基本功能
但其中还未添加入用户登录功能，暂时只完成了用户界面的编写

## 首先需要在server文件夹下运行本地服务器（才可以使用用户登陆功能）
node app.js

## Project setup
```
npm install
```

### 运行程序
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```
